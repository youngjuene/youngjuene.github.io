## How to Submit

Add your card to this page with a pull request:

1. Fork [`youngjuene/youngjuene.github.io`](https://github.com/youngjuene/youngjuene.github.io)
   and edit `courses/docs/module2/dashboard.md`.
2. Copy an example card and replace it with your team name, title,
   description, members, and links (project page, video, code, slides — host
   media on your own drive/YouTube/Vimeo and link it).
3. Open a pull request titled `[Module 2] Team <name> — <work title>`.
   It will be reviewed and merged before the next session.

!!! tip
    No GitHub experience? Send the same information (title, description,
    members, links) by email to youngjun.choi (at) kaist.ac.kr and it will be
    added for you.
